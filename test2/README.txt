## 주요 업무
- 클라이언트로부터 이름, 성별, 키, 몸무게를 입력받아 서버로 전송한다.
- 이름, 성별, 키, 몸무게, 표준체중, bmi가 계산되어 
  클라이언트로 재전송한다.
- 결과를 전송받은 클라이언트는 
     이름, 성별, 키, 몸무게, 표준체중, bmi를 화면에 출력한다.

## 필요한 파일
1) MainController / FitnessController

2) index.html -> 정보를 입력받는다.
   bmiResult.html -> 최종 결과를 출력할 파일
   
3) DTO 파일  : Fitness.java
